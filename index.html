Adding your data to a dashboard

If you would like your organisation included on one of the dashboards above you should first make sure that you create a CSV file with column headings that match those in our data standard. We're making a diversity data builder tool to help. Once you have a valid file it should be published openly on the web at a fixed URL under the Open Database Licence (ODbL). Once your data is published openly, it needs to be added to the appropriate dashboard index e.g. Leeds City Region index. If you have a Github account you can create a pull request directly or you may contact Open Innovations to get it added manually.
